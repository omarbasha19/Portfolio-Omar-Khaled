## Selected Projects

### OmniRAG Assistant

A multimodal AI assistant built with Python, Streamlit, Ollama, FAISS, LLaVA, and RAG.

Key highlights:

* Supports multiple input types including PDF, CSV, arXiv, image, audio upload, and live microphone input
* Implements document question answering using embeddings, FAISS retrieval, and local LLM inference
* Includes image understanding, audio transcription, text-to-speech, summarization, and PDF chat export

Repository:

https://github.com/omarbasha19/OmniRAG-Assistant-Multimodal-Document-QA-Chatbot-Streamlit-Ollama-FAISS-LLaVA-

### ReviewPulse: Amazon Sentiment Analytics

A real-time sentiment analytics pipeline built with Python, Spark MLlib, Kafka, MongoDB, and Plotly Dash.

Key highlights:

* Processes large-scale Amazon review data for sentiment analysis
* Trains and compares machine learning models for text classification
* Stores predictions in MongoDB and visualizes insights through an interactive dashboard

Repository:

https://github.com/omarbasha19/ReviewPulse-Real-Time-Amazon-Sentiment-Analytics

### Driver Monitoring System

A computer vision system built with Python, TensorFlow, and Gradio for distracted driver behavior classification.

Key highlights:

* Classifies driver behavior from in-car images
* Uses deep learning models for image classification
* Includes confidence scoring, risk assessment, prediction logging, and an interactive interface

Repository:

https://github.com/omarbasha19/Driver-Monitoring-System

## Run Locally

Open the project folder and run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

You can also open the `index.html` file directly in a browser.
