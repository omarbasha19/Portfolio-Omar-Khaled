# Omar Khaled Portfolio Website V6 Premium Responsive

A clean, fast, responsive portfolio for Omar Khaled.

## What changed in V6

- Stronger AI Engineer and Data Scientist hero statement.
- Improved icon alignment for AASTMT, LinkedIn, GitHub, ORCID, Google Scholar, and ResearchGate.
- Smaller, cleaner typography.
- Better desktop and mobile layout.
- No exact repository count, so the site needs less frequent updates.
- No duplicated project entries in the archive.
- Lightweight animations using IntersectionObserver.
- No heavy external libraries.

## Run locally

Open `index.html` directly, or run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Edit content

Update content in:

```text
js/data.js
```

## Assets

Main assets are stored in:

```text
assets/
```

Replace the academy logo, profile photo, or social icons without changing the code if you keep the same file names.
